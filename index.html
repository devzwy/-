工作经历

上海可零网络科技有限公司（2019年05月-今）

开发工具： AndroidStudio + MVVM + Jetpack + Html + 第三方框架 各店 项目使用了到大量第三方框架，主要业务模块：直播带货、短视频播放、 IM即时通讯。该项目原生部分为个人独立开发。重构的初衷：原项目未使用任何设计模式， 随着项目的日益增大，维护起来很不方便。另方面由于项目代码质量不高，大面积的内存泄 露导致软件频繁闪退、卡死、加载Html部分页面卡顿严重，用户反馈不好用。重构版本使用 Kotlin + LiveData + MVVM + Coroutine + Retrofit + OkHttp 等。为了优化 Html的加载速度，我这边选用了腾讯X5的浏览器内核，提升加载效果很明显。 各店更新工具 由于各店项目中使用到了大量的html的代码，这些html代码更新比较频繁，导致更新html部分代码时需要 App重新打包后发布更新apk(原项目html代码存在assets下)，考虑到这样给用户的体验不是很好，我决定 将原项目加载本地html的方式由Assets更改至SD卡中，让app动态的去下载html代码，所以各店更新工具 就诞生了。该工具使用Python + Pyqt5编写，可编译Mac、Windows双平台运行。工具主要由三种更新模 式，模式一：Apk全量更新，使用工具上传apk，填写发布信息，app端检测到由更新时只需下载apk安装即 可。模式二：Html全量更新，工具上传最新的Html代码，App检测更新并下载替换本地文件即可。模式三： Html补丁更新。该模式主要解决了模式二下全量更新html时用户下载慢的问题，用户填写需要更新的包和发 布信息，工具自动与线上html代码做比对，找出需要更新的文件并生成配置文件，app端检测到更新时替换部 分文件即可。工具为个人独立开发。
